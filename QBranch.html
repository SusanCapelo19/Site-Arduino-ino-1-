Componentes Utilizados

Arduino Uno Blackboard (R3)

Quantidade: 1

Placa de desenvolvimento baseada no microcontrolador ATmega328P, compatível com a plataforma Arduino. Possui 14 pinos digitais de I/O (6 PWM), 6 entradas analógicas, interface USB, cristal de 16 MHz, botão de reset e conectores de alimentação.

Código de Referência:A000066

Onde encontrar

Módulo de LED MAX7219 32x8

Quantidade: 1

Display de matriz de LED controlado pelo chip MAX7219, capaz de controlar até 64 LEDs (8x8). O modelo 32x8 é uma junção de 4 matrizes 8x8, permitindo maior espaço visual. Comunicação via SPI com o Arduino.

Código de Referência:MAX7219 Dot Matrix Module 4-in-1 32x8

Onde encontrar

Módulo de Microfone MAX4466

Quantidade: 1

Microfone eletreto acoplado ao amplificador operacional MAX4466, ideal para capturar sons com boa sensibilidade e pouco ruído. Possui potenciômetro para ajuste de ganho.

Código de Referência:MAX4466 Electret Microphone Amplifier Module

Onde encontrar

Breadboard (Protoboard)

Quantidade: 1

Placa de ensaio usada para montar circuitos eletrônicos temporariamente, sem a necessidade de solda. Permite conexões rápidas e reutilizáveis entre os componentes.

Quantidade: 16

Biblioteca para Arduino que implementa a Transformada Rápida de Fourier (FFT - Fast Fourier Transform). Ela permite transformar um sinal no domínio do tempo (como o captado por um microfone) para o domínio da frequência.

Biblioteca SPI

SPI (Serial Peripheral Interface) é um protocolo de comunicação serial síncrona que permite a troca de dados em alta velocidade entre um dispositivo mestre e um ou mais dispositivos escravos, utilizando uma arquitetura full-duplex e uma configuração de barramento com quatro linhas principais: MISO (Master In Slave Out), MOSI (Master Out Slave In), SCK (Serial Clock) e SS/CS (Slave Select/Chip Select). É uma biblioteca interna e nativa da IDE Arduino, não precisa de download externo. Abaixo você pode ter acesso a documentação da biblioteca, ensinado como usá-la, entre outras funcionalidades.
